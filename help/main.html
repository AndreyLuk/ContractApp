Программа "Договоры" - это небольшое клиент-серверное приложение, написанное на языке программирования C++ с использованием каркаса приложений Qt.

Автор программы - Андрей Владимирович Лукашенко.
Кто угодно может использовать эту программу или ее часть при выполнении условий:
1) Принимая то, что программа предоставлена как есть без каких либо гарантий.
2) Ссылки на автора недопустимы.
Часть технических решений, примененных в программе, может быть использована в промышленных программах, а часть - нет.

В качестве СУБД применяется PostgreSQL.
Скрипт создания схемы данных можно посмотреть по этой ссылке .
В схеме создаются четыре таблицы с отношениями "один ко многим", а также индексы и триггеры.
В скрипте используются SQL и PL/pgSQL.
В скрипте также производится заполнение таблиц тестовыми данными.

Программа состоит из следующих частей:

  • Начало работы
  • Ведение договоров
  • Отчет о договорах
  • Справочник типов договоров
  • Справочник состояний договоров
  • Справочник контрагентов
